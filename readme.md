# Synerdrone

A hardware based synergy client

> [!IMPORTANT]
> Work in progress

## Objectives
- [X] Keepawake mode - moves the mouse cursor randomly to keep host awake
- [ ] synergy client mode - connects to a synergy server and relays mouse and keyboard to host


## Suported hardware
- [Lilygo T-Dongle-S3](https://wiki.lilygo.cc/products/t-dongle-series/t-dongle-s3/)

## Setup
> [!WARNING]
> Please follow device's documented settings for compilation and programming

Currently the only way to build and program the dongle is using the Arduino IDE.

## Usage
Pushing the boot button of the dongle starts/stops moving the mouse randomly
